# create-cursor-skill

Create Agent Skills — folder structure, SKILL.md, frontmatter, .mdc rules ([agentskills.io](https://agentskills.io/)).

**What it does:** Plan mode first; skill vs rule; SKILL.md, reference.md, examples; frontmatter format.

## Quick install

```bash
git clone https://github.com/mariocosttaa/my-agent-skills.git && cp -r my-agent-skills/create-cursor-skill ~/.cursor/skills/
```
