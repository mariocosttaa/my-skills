# Version snapshots

This folder holds released versions of this skill (e.g. `v1.3/`, `v1.4/`).

- **Latest:** use the skill root (`../`).
- **Specific version:** copy `versions/vX.Y/` to `~/.cursor/skills/<skill-name>/`.

See [repo versioning](https://github.com/mariocosttaa/my-agent-skills) for workflow.
