# qa-agent

QA engineering: browser testing, 10 user profiles, test resources, structured reports. **Requires Browser MCP** (mario-playwright-mcp or browsermcp).

**What it does:** Navigate, snapshot, click, screenshot via MCP; user profiles; task plan before testing; reports in `QA-AGENT/<project>/test/`.

## Quick install

```bash
git clone https://github.com/mariocosttaa/my-agent-skills.git && cp -r my-agent-skills/qa-agent ~/.cursor/skills/
```
