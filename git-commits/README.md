# git-commits

Commit messages and branch flow: Conventional Commits, file grouping, stash/checkout workflow.

**What it does:** Format `type: description` + bullets; grouping per commit; stash → checkout → pop → push; never adds Made-with trailer.

## Quick install

```bash
git clone https://github.com/mariocosttaa/my-agent-skills.git && cp -r my-agent-skills/git-commits ~/.cursor/skills/
```
